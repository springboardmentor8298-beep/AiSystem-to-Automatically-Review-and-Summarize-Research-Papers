# AI System to Automatically Review and Summarize Research Papers

## Overview
This project aims to automatically analyze, review, and summarize research papers using NLP and AI techniques.  
It helps users quickly understand the key ideas, contributions, and findings of academic papers.

## Features
- Upload and analyze research papers
- Automatic text extraction
- AI-based summarization
- Key point and keyword extraction

## Tech Stack
- Python
- Natural Language Processing (NLP)
- Transformer-based models (future scope)

## Setup Instructions
1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Run the summarization script

## Future Enhancements
- PDF support
- Section-wise summaries
- Citation extraction
